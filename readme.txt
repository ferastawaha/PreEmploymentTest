To run the project please follow theses instructions:
-choose java 1.8 in your project (if you are using intellijIdea go to file the project structure and chose jdk1.8)
-i'm using maven to manage our dependencies like junit 
-for unit test i'm using junit jupiter so to run the tests go to test folder and run the test for each class 
-for every task there is a class which contain a main method and other methods containing the logic 
-First you need to run the main method in the class main then you choose a number of the task and you enter the inputs.
